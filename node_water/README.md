# NODE Water

**ESP32-C3** - Управление водой

## Функции

- Насосы подачи/слива
- Соленоидные клапаны (3 зоны)
- Датчик уровня воды (опционально)

## Распиновка

| GPIO | Назначение |
|------|------------|
| 1 | Насос подачи |
| 2 | Насос слива |
| 3 | Клапан зона 1 |
| 4 | Клапан зона 2 |
| 5 | Клапан зона 3 |
| 6 | Датчик уровня |

## Сборка

```bash
cd node_water
idf.py set-target esp32c3
idf.py build
idf.py -p COM8 flash monitor
```
