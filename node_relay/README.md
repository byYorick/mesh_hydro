# NODE Relay

**ESP32** - Управление климатом

## Функции

- Линейные актуаторы форточек (2 шт)
- Вентилятор
- LED свет с PWM диммированием

## Распиновка

| GPIO | Назначение |
|------|------------|
| 1 | Актуатор форточки 1 |
| 2 | Актуатор форточки 2 |
| 3 | Вентилятор |
| 7 | LED свет (PWM) |

## Сборка

```bash
cd node_relay
idf.py set-target esp32
idf.py build
idf.py -p COM7 flash monitor
```
